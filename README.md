# PolyEdge Project Structure

- `data/raw/` - Raw data from APIs 
- `data/processed/` - Cleaned, formatted data ready for training
- `training/scripts/` - Fine-tuning scripts
- `training/checkpoints/` - Saved model weights
- `backend/` - API server code
- `frontend/` - Next.js app
- `docs/` - Documentation
- `scripts/` - Utility scripts
